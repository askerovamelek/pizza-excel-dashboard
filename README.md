# Pizza Dashboard

## Layihə Haqqında
Bu layihə **Pizza Dashboard**-u qurur. SQL sorğuları ilə satış məlumatlarını analiz edir və Excel-də vizual dashboard yaradır. Məqsəd pizza satışlarını izləmək və analiz etməkdir.

## İstifadə Olunan Texnologiyalar
- SQL – Məlumatları təhlil etmək üçün  
- Excel – Dashboard yaratmaq və vizualizasiya üçün  

## Layihənin Funksiyaları
1. Ümumi gəlir (Total Revenue)  
2. Orta sifariş dəyəri (Average Order Value)  
3. Ümumi satılan pizza sayı (Total Pizzas Sold)  
4. Sifarişlərin ümumi sayı (Total Orders Count)  
5. Orta pizza sayı sifariş başına (Average Pizzas per Order)  
6. Sifariş sayı gün və saat üzrə (Orders by Day & Hour)  
7. Gəlir və satış payı pizza kateqoriyası və ölçüsünə görə  
8. Ən çox və ən az satılan pizzalar  
9. Excel dashboard ilə vizual analiz  

## Fayllar
- `pizza_dashboard.sql` – SQL sorğuları  
- `Pizza_Dashboard.xlsx` – Excel dashboard
